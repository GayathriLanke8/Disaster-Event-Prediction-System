# 🌍 Disaster Event Prediction System

## 📌 Project Overview

The Disaster Event Prediction System is a Machine Learning project developed to predict disaster events using historical environmental and disaster-related data. The project follows a complete end-to-end machine learning workflow, including data preprocessing, exploratory data analysis, model building, evaluation, and prediction.

The system helps identify potential disaster events by learning patterns from historical data and classifying whether a disaster is likely to occur.

---

## 🚀 Results

✔ Processed and analyzed **20,000+ records** with **13 features**

✔ Built a complete Machine Learning pipeline for disaster prediction

✔ Trained and compared **5 Machine Learning models**

✔ Achieved up to **100% testing accuracy**

✔ Identified the best-performing model through comparative evaluation

